Sim, você está absolutamente correto! Para fazer a conversão funcionar bem em ambos os sentidos (de qualquer unidade de origem para qualquer unidade de destino), o ideal é seguir esses passos:

Receber a unidade de origem e a unidade de destino: Primeiro, o usuário escolhe qual unidade quer converter (origem) e para qual unidade deseja a conversão (destino).

Converter para uma unidade de referência (padrão): A ideia é converter a unidade de origem para uma unidade de referência padrão, como metros no caso de comprimento. Isso simplifica o processo, pois cada unidade só precisa de um fator de conversão para o metro, em vez de para todas as outras.

Converter da unidade padrão para a unidade de destino: Uma vez que você tem o valor na unidade padrão (metros, por exemplo), basta convertê-lo para a unidade de destino.

Exemplo Prático Sem Código
Imagine que você quer converter pés (feet) para centímetros (cm). O processo seria:

Unidade de Origem: Pés (feet).
Unidade de Destino: Centímetros (cm).
Converter para a Unidade Padrão (metros): Converta pés para metros usando o fator de conversão (1 pé = 0,3048 metros).
Converter da Unidade Padrão para a Unidade de Destino: Agora que o valor está em metros, converta para centímetros (1 metro = 100 centímetros).
Esse fluxo simplifica muito o processo, pois cada unidade de comprimento só precisa ter uma conversão para e de metros.
